## Packages
recharts | For visualizing network statistics (bitrate, latency) graphs
framer-motion | For smooth animations of memory log updates and UI transitions
date-fns | For timestamp formatting in logs and chat
clsx | Utility for conditional classes
tailwind-merge | Utility for merging tailwind classes

## Notes
- "StreamChat" simulation logic will run client-side for network stats to ensure 60fps smoothness.
- Video player uses standard HTML5 <video> tag.
- Dark mode is the default and only theme for this cyber-punk aesthetic.
- Chat completions hit /api/chat/completions.
